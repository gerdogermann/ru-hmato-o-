Python (Programmeerimise keel) on kõrgetasemeline, tavakasutusele orienteeritud programmeerimise keel. Disaini filosoofia näeb ette koodilugevust kasutades taandeid. Python on dünaamiline ja prügikorjav. Pythonis on tugi struktureeritud, objektorienteeritud ja funktsionaal programmeerimisele. Python loodi 1991, loojaks oli Guido van Rossum. Esimene projekti versioon oli 0.9.0. 2000. aastal tuldi välja versiooniga Python 2.0, millel oli paljuseid tänapäeval kasutatavaid programmistruktuure. Python 3.0 tuli välja 2008. aastal, ning oli nii meeletu muutus, et see polnud osaliselt “backward-compatible” vanemate versioonidega. Python 2 seisati 2020. aastal.


Python tuum-filosoofia on kokkuvõetud dokumendis “The Zen of Pyhton”, kus on kirja pandud mõtteterad:

  • Beautiful is better than ugly
  • Explicit is better than implicit
  • Simple is better than complex
  • Complex is better than complicated
  • Readability counts

Python on loodud kasutama erinevaid mooduleid enda andmebaasist. Van Rossumil oli visioon väikse tuumaga keelest, suure standardse laekaga.


Süntaks
