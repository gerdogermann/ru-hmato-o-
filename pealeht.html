Python

Python (Programmeerimise keel) on kõrgetasemeline, tavakasutusele orienteeritud programmeerimise keel. Disaini filosoofia näeb ette koodilugevust kasutades taandeid. Python on dünaamiline ja prügikorjav. Pythonis on tugi struktureeritud, objektorienteeritud ja funktsionaal programmeerimisele. Python loodi 1991, loojaks oli Guido van Rossum. Esimene projekti versioon oli 0.9.0. 2000. aastal tuldi välja versiooniga Python 2.0, millel oli paljuseid tänapäeval kasutatavaid programmistruktuure. Python 3.0 tuli välja 2008. aastal, ning oli nii meeletu muutus, et see polnud osaliselt “backward-compatible” vanemate versioonidega. Python 2 seisati 2020. aastal.

Python tuum-filosoofia on kokkuvõetud dokumendis “The Zen of Pyhton”, kus on kirja pandud mõtteterad:

  • Beautiful is better than ugly
  • Explicit is better than implicit
  • Simple is better than complex
  • Complex is better than complicated
  • Readability counts

Python on loodud kasutama erinevaid mooduleid enda andmebaasist. Van Rossumil oli visioon väikse tuumaga keelest, suure standardse laekaga.

Pythoni disaini filosoofiast tulenevalt on paljud käsusõnad lihtsasti tuletatavad inglisekeelsed sõnad. Erinevalt teistest keeltest ei kasutata Pythonis väga loogelisi sulge. Erinevate blokkide eraldamiseks kasutatakse tühikuid.

Tüüp Kirjeldus
Bool Boolean väärtus.
Complex Kompleksarv, reaal ja imaginaararvudega.
dict Sõnastik, kus väärtusi hoitakse võtmetes.
Moodul Kirjeldus
NumPy Pythoni programmeerimiskeele raamatukogu, mis lisab tuge suurtele, mitmemõõtmelistele massiividele ja maatriksitele, koos suure hulga kõrgetasemeliste matemaatiliste funktsioonidega, et neil massiividel töötada.
TensorFlow Avatud lähtekoodiga masinõppe tarkvarateek, mis kasutab arvutuste tegemiseks andmevoo graafe. Teegi peaeesmärk on seotud masinõppe ja tehisnärvivõrkude uurimisega, kuid see on piisavalt üldine rakendamiseks muudes valdkondades.
PyGame Pythoni teek, mille abil saad lihtsalt luua erinevaid arvutimänge. See võimaldab panna jalg mängudemaailma ukse vahele ning erendada oma programmeerimise oskusi.
